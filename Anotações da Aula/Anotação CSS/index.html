O CSS é uma linguagem de formatação e estilização.

É uma linguagem de folha de estilo em cascata.

É uma linguagem que define a aparência de um documento HTML.

Regras de estilização:

  • Seletores
  • Corpo
  • Propriedade
    • Valor

elemento {
propriedade: valor;
}

Ou...

seletor1, seletor2 {
propriedade: valor;
}

Propriedades de áreas e margens:

  • margin-left
  • margin-right
  • margin-bottom
  • margin-top
  • margin
